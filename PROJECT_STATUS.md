# 프로젝트 현황 보고서

## 현재 상태
- 도서 등록, 검색, 상태 필터, 정렬, 상세 조회가 동작합니다.
- 독서 로그는 페이지 범위 검증과 함께 저장되며 진행률을 자동으로 갱신합니다.
- 리뷰와 별점은 상세 화면에서 저장/수정할 수 있습니다.
- 대시보드는 읽는 중인 책, 완독한 책, 누적 페이지를 보여줍니다.
- 테마 토글은 라이트/다크 모드를 지원하며 `localStorage`에 저장됩니다.
- 스토어는 입력 검증, 상태 전이, 구버전 데이터 정규화를 담당합니다.

## 이번에 보강된 항목
- 스토어 액션을 결과 반환형으로 통일하여 화면에서 인라인 오류를 처리할 수 있게 함
- `coverUrl` URL 입력 지원 및 카드/상세의 표지 폴백 UI 추가
- 목록 정렬 기능 추가: 최신 등록순, 제목순, 진행률순, 상태순
- 잘못된 책 입력, 잘못된 로그 입력, 잘못된 별점 입력에 대한 사용자 피드백 추가
- 라이트/다크 테마 디자인과 모바일 레이아웃 정리
- 테스트를 주요 사용자 시나리오 중심으로 재작성
- Playwright 기반 브라우저 E2E 테스트 추가

## 남은 과제
- 외부 도서 검색 API 연동
- 독서 캘린더/히트맵 시각화
- 클라우드 동기화 또는 백엔드 연동
- CI에 Playwright E2E 실행 추가
