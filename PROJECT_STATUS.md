# 프로젝트 현황 보고서

## 현재 상태
- 도서 등록, 검색, 상태 필터, 정렬, 상세 조회가 동작합니다.
- 독서 로그는 페이지 범위 검증과 함께 저장되며 진행률을 자동으로 갱신합니다.
- 리뷰와 별점은 상세 화면에서 저장/수정할 수 있습니다.
- 태그/키워드를 책에 추가·삭제할 수 있으며, 태그로 검색이 가능합니다.
- 대시보드는 읽는 중인 책, 완독한 책, 누적 페이지를 보여줍니다.
- 대시보드에 GitHub 잔디 스타일의 52주 독서 활동 히트맵이 표시됩니다.
- 대시보드에 최근 12개월 읽은 페이지/완독 권수 월별 바 차트가 표시됩니다.
- 대시보드에서 JSON 백업 내보내기와 미리보기 기반 복원(병합/덮어쓰기)이 가능합니다.
- 전역 UI는 `DESIGN.md` 기준의 Raycast 스타일 다크 디자인 시스템으로 정리되어 있습니다.
- 헤더, 히어로, 카드, 버튼, 입력 필드가 공통 surface token과 shadow 계층을 공유합니다.
- 책 삭제 후 5초 동안 복구할 수 있는 undo 흐름이 있습니다.
- 테마는 `system / light / dark` 3상태를 지원하며 기본값은 시스템 설정입니다.
- 테마와 책 데이터는 IndexedDB(Dexie)에 저장되며, 기존 `localStorage` 데이터는 첫 실행 시 자동 마이그레이션됩니다.
- 스토어는 입력 검증, 상태 전이, 구버전 데이터 정규화, 백업/복원, 테마 해상(resolution)을 담당합니다.
- GitHub Actions CI에서 단위 테스트와 Playwright E2E 테스트가 병렬로 실행됩니다.
- 현재 검증 기준은 Vitest 42개 테스트와 Playwright 13개 시나리오입니다.
